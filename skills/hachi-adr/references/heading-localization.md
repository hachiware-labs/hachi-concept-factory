# hachi-adr Heading Localization

When `output_locale` is Japanese, use Japanese headings and field labels.

## Heading Map

- `Metadata` -> `メタデータ`
- `Context` -> `背景`
- `Assumption Ledger` -> `前提台帳`
- `Decision` -> `決定`
- `Rationale` -> `理由`
- `Alternatives Considered` -> `検討した代替案`
- `Consequences` -> `結果`
- `Risks` -> `リスク`
- `Mitigations` -> `緩和策`
- `Implementation Notes` -> `実装メモ`
- `Verification Notes` -> `検証メモ`
- `Follow-up Tasks` -> `フォローアップタスク`
- `Related Documents` -> `関連文書`
- `ADR Decision` -> `ADR判断`
